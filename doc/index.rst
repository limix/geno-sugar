===========
Geno Sugar!
===========

.. Attention::

  While this can be used as a standalone module, this module has been designed to
  be a building block of LIMIX2_. LIMIX2 is a python module that makes available a
  large set of tools for genetic studies, including statistical methods for GWAS,
  interaction testing and variance component estimation.

.. _LIMIX2: https://limix.readthedocs.io/en/2.0.0/index.html
